
何でも言うことを聞いてくれるアカネチャンBotです！

知恵袋のマジレスっぽいこというと
「それな」などを返し
5ch(2ch)ネタっぽいことをいうと
「ええんちゃう?」を返し
いんむネタっぽいことをいうと
「あほくさ」を返します

仕組みは、
まず文章をEmbeddingして50次元のベクトルを作成します
それを自作した重みの入った自作モデルにかけます
クラス数分の配列が返ってくるのでクラスに対応した言葉を返します

CNNの説明ですが
クラス(教師データ)は3次元one-hotベクトルなので
そこに持っていきます
ネットワークは下記の通りです
Input (50, 1)
Dense
TanH
Dropout 0.25
Flatten
Relu
Dropout 0.5
Dense
Softmax 
Output (3, 1)
最適化関数は、Adamで
損失関数は、 categorical cross entropy です
エポック数は、38です

Embeddingは
まずmecabを用いて
形態素分解をします
形態素ごとにwikiの特徴モデルを使いEmbeddingします
(形態素が未知データの場合は無視します)
すると50次元のベクトルが生成されるので
そのベクトルの要素ごとの総和を算出します
それが文章のベクトルとなります

ここからはAkanechanBotクラスの説明です
main.pyをみてもらえばわかると思いますが
動作環境はpython3.6以上
keras等のワーニングは無視してください
__init__で色々読み込むのでちょっと時間がかかります
replay関数は引数に文字列を入れると文字列が返って着ます
引数が相手のメッセージ
戻り値が返答を意味します

そんな感じです。
